Sample behavioral waveforms for design file st6.vhd

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design st6.vhd. The design st6.vhd is a 6 bit up modulus 60 counter. The counter has an asynchronous clear input.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.

Fig. 2 : Wave showing counter cout and/or modulus operation.

The output port cout will be asserted at the completion of count sequence. The counter counts till the modulus value 59.
